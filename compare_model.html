★ Verdict ★

The PPO v10 model with GRU memory and Self-Imitation Learning consistently reaches Level 10 with a 134K average score in a game far more complex than standard RL benchmarks. The GRU side-channel adds +1.7 average levels over vanilla PPO on the same features, demonstrating that temporal memory is critical for handling the game's dynamic threats: homing missiles, 8 Monster2 movement patterns, kamikaze dives, and escalating bullet barrages.

The dual-model evolutionary selection (F4 in-game) lets both V10 and Vanilla PPO compete at each decision step — the action from the more confident model wins. CPU-based lookahead simulation (F3) provides an additional safety net by simulating actions forward 8 ticks and vetoing life-threatening choices.
